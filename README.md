## Artspeak : Language of art

A language to create art and animation with a easy and intuitive syntax and their semantics.

### Language Design

**Basic Blocks**

1. Objects that can be used in the canvas.
    - Basic Shapes - circle, rectangle, square
    - Lines and points
2. Function to manipulate Objects
    - canvas.add : add object to the scene
    - (?) canvas.remove : remove a already added object, might be helpful in the repl
